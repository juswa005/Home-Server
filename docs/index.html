Self-Hosted Stack

How I Replaced Cloud Services With My Home Server

This repository documents an Ubuntu home server that replaces common hosted services with a small, practical self-hosted stack built around Docker, Tailscale, and a few focused apps.

Ubuntu 24.04.4 LTS Docker-based Remote access with Tailscale
Check GitHub Repo

Overview

What the server handles

Nextcloud

Personal cloud storage and file sync.

Immich

Photo and video backup with a modern gallery.

Navidrome

Music streaming for a personal library.

Gitea

Self-hosted Git hosting and web UI.

Portainer

Docker management and container visibility.

Host Services

SSH, UFW, Docker, and Tailscale on Ubuntu.

Cloud Replacements

Hosted tools this setup replaces

Dropbox / Google Drive

Replaced by Nextcloud for personal file storage and sync.

Google Photos

Replaced by Immich for media backup and browsing.

Hosted music services

Replaced by Navidrome for streaming my own library.

GitHub / GitLab

Partly replaced by Gitea for personal project hosting.

Hosted admin dashboards

Replaced by Portainer for container management.

Host Snapshot

  • Hostname: ubuntu-server
  • OS: Ubuntu 24.04.4 LTS
  • Kernel: Linux 6.8.0-107-generic

  • Firewall: ufw enabled
